# Full Stack Landing Page & Admin Panel

A full-stack web application that includes a modern landing page and an admin panel for managing projects, clients, contact form submissions, and newsletter subscriptions.

---

## Features

### Landing Page
- Responsive and modern UI
- **Our Projects**
  - Fetches projects from backend
  - Displays project image, name, description
  - Includes a dummy "Read More" button
- **Happy Clients**
  - Displays client testimonials from backend
  - Includes image, name, designation, and description
- **Contact Form**
  - Fields:
    - Full Name
    - Email Address
    - Mobile Number
    - City
  - Submits data to backend
- **Newsletter Subscription**
  - Email input with subscribe button
  - Saves subscribed email to backend

---

### Admin Panel
- Secure dashboard for managing all data

#### Project Management
- Add projects with:
  - Image upload
  - Project name
  - Project description
- View all projects

#### Client Management
- Add clients with:
  - Image upload
  - Name
  - Description
  - Designation
- View all clients

#### Contact Form Submissions
- View all submitted contact details:
  - Name
  - Email
  - Mobile Number
  - City

#### Newsletter Subscribers
- View all subscribed email addresses

---

## Bonus Feature
- Image cropping during image upload in admin panel
- Images are cropped to a fixed ratio before storing
